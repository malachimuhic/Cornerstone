# Cornerstone
Machine Learning library written in C

This is a custom ML library written from scratch to recognize handwritten digits.It includes custom memory management, tensor/matrix math, gradient calculations, and activation functions.

### Features
- Handwritten digit recognition
- Custom arena allocator for memory management
- Tensor and matrix operations
- Forward pass logic
- Gradient calculations
- Activation functions

### Purpose
The goal of this project was to better understand how machine learning works at a lower level. By building the system in C, I was able to explore how neural networks handle memory, numerical operations, and model training without relying on libraries like TensorFlow or PyTorch.
